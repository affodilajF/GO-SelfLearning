# POINTER -------------- 

### Pass by Value
  By default di Go semua variabel itu passing by value, bukan by reference (menuju lokasi memori data).
  Artinya jika kita mengirim variable ke dalam function, method, atau variabel lain, yang dikirim adalah duplikasi value nya.

  Data di address1 akan di copy ke address2. Perubahan berdampak individu, tidak saling mempengaruhi variabel lain. 
  ![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/03d40bbd-2194-4f2b-97bd-ed039c32d669)

  ![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/6448d3d6-7baf-41cf-9511-77a7d614bf0d)

### POINTER : solusi biar bisa pass by ref aja. (OPERATOR &) 
  Pointer adalah kemampuan membuat reference ke lokasi data di memory yang sama tanpa menduplikasi data yang suah ada.

  How cara membuatnya?
  - Operator &
    Untuk membuat sebuah variabel dengan nilai pointer ke variabel lain.
    ![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/bff77ce4-4eef-4355-a7dc-a9b5565d944d)
    ![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/175683eb-8300-4bff-9e82-3fe838dded0e)

# ASTERISK OPERATOR ------------------
Operator bintang. 

- TANPA OPERATOR *
Misal address2 = &address1, artinya address2 adalah sebuah pointer. Jika ingin address2 menjadi pointer dari referensi selain address1, 
caranya adalah address2 = &(ref baru). (Menggunakan &). 

Efeknya, address1 dan addres2 tidak memiliki ikatan apapun lagi. 

![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/d0d94466-43ce-47b3-8405-36aabece4d79)

![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/46519083-d95f-457c-8656-3075f8f31ada)

- DENGAN OPERATOR *
Agar ikatan address1 dan address2. Jadi ketika address2 ganti data, address1 juga ikut keganti.

![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/378b043a-ac38-40eb-907f-611d181693f7)

![image](https://github.com/affodilajF/GO-SelfLearning/assets/130672181/fbbdca46-5e74-43b4-9eae-e4d3a468ef63)
